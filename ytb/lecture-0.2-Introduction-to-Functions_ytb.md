в—Ҹ [09:40]. The Vertical Line Test 
     в—ү Reminder: in tables, repeated inputs with different outputs mean it is not a function
     в—ү Explanation of the Vertical Line Test
          в—Ӣ A graph is a function if and only if every vertical line intersects it at most once
     в—ү Graphic examples
          в—Ӣ A diagonal line: it is a function
          в—Ӣ A parabola: it is a function (though not one-to-one)
          в—Ӣ A graph that passes the test even if it has an empty section (e.g., 1/рқ’ҷ)
               в–  Not every input must have an output, but if it does, it must be unique
          в—Ӣ A graph that fails the Vertical Line Test (one input with multiple outputs) is not a function
